Sistema de control de Sesiones de Laboratorio

El laboratorio de cómputo ofrece a los estudiantes de las carreras de informática- Sistemas la opción de realizar prácticas, en dichos ambientes con equipos altamente actualizados para el desarrollo de prácticas de diferentes materias.

Este sistema permite el control de las sesiones de diferentes materias que desarrollan prácticas y que ayudan al estudiante, a administrar prácticas, tareas para fortalecer y mejorar sus habilidades de práctica en dicha materia.

Apoyado de una evaluación que permite ver el rendimiento de un estudiante al momento de realizar una práctica.
